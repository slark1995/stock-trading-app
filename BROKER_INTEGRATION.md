# 券商集成指南

本文档说明如何将您的同花顺账户与A股自动交易系统集成。

## 📋 前置条件

### 1. 安装必要的Python库

```bash
pip3 install easytrader tushare akshare
```

### 2. 同花顺账户信息

- 同花顺用户名（手机号或邮箱）
- 同花顺密码
- 券商代码（开源证券、东吴证券等）

## 🔗 连接账户步骤

### 步骤1：启动应用

```bash
# Windows
start-local.bat

# Linux/Mac
bash start-local.sh
```

### 步骤2：点击"连接账户"按钮

在仪表盘上点击绿色的"🔗 连接账户"按钮。

### 步骤3：填写账户信息

1. **选择券商**：从下拉菜单中选择您的券商
   - 开源证券
   - 东吴证券
   - 华泰证券

2. **输入用户名**：您的同花顺账户用户名

3. **输入密码**：您的同花顺账户密码

4. **点击"连接账户"**

### 步骤4：验证连接

连接成功后，您将看到：
- ✅ 账户余额
- ✅ 总资产
- ✅ 可用资金
- ✅ 股票市值
- ✅ 冻结资金

## 📊 功能说明

### 已连接后可用的功能

#### 1. 查看持仓
- 点击"📈 查看持仓"查看您在券商的实时持仓
- 显示每只股票的成本价、现价、盈亏等信息

#### 2. 创建策略
- 点击"📊 管理策略"创建自动交易策略
- 基于技术指标（MA、MACD、RSI等）自动生成交易信号

#### 3. 自动交易
- 系统根据策略规则自动扫描市场
- 当满足条件时自动下单买入或卖出
- 所有交易记录保存在"📝 交易记录"中

## 🔒 安全建议

### 1. 账户信息保护
- ✅ 账户信息仅在本地使用
- ✅ 不会被上传到服务器
- ✅ 密码不会被记录或保存

### 2. 网络安全
- 仅在安全的网络环境下使用
- 不要在公共WiFi上连接账户
- 定期更改同花顺密码

### 3. 风险管理
- 设置合理的止损止盈规则
- 不要设置过激进的策略
- 定期检查交易记录

## 🐛 常见问题

### Q: 连接失败，提示"登录失败"
**A**: 
1. 检查用户名和密码是否正确
2. 确保选择了正确的券商
3. 检查网络连接
4. 尝试在同花顺软件中登录验证账户是否正常

### Q: 无法获取持仓信息
**A**:
1. 确保已成功连接账户
2. 检查您的账户是否有持仓
3. 查看浏览器控制台是否有错误信息

### Q: 如何修改账户信息
**A**:
1. 点击"登出账户"按钮
2. 重新填写新的账户信息
3. 点击"连接账户"
